# PRODIGY_DS_01 - Data Visualization using Bar Chart and Histogram

### 📌 Objective
The objective of this project is to create visualizations that represent the distribution of categorical and continuous variables using Python. The analysis helps in understanding the characteristics of the dataset through graphical representations.

---

## 📂 Dataset

- **Dataset:** Titanic Dataset
- **Source:** Kaggle Titanic - Machine Learning from Disaster
- **Files Used:**
  - train.csv

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Visualizations Created

### 1. Gender Distribution (Bar Chart)
- Displays the number of male and female passengers.
- Helps understand the distribution of passengers by gender.

### 2. Passenger Class Distribution (Bar Chart)
- Shows the number of passengers in each travel class (1st, 2nd, and 3rd).
- Highlights the proportion of passengers across different classes.

### 3. Age Distribution (Histogram)
- Illustrates the frequency distribution of passenger ages.
- Useful for identifying the most common age groups.

### 4. Fare Distribution (Histogram)
- Displays the distribution of ticket fares.
- Reveals the presence of outliers and fare variations.

---

## 📈 Key Insights

- The majority of passengers were male.
- Most passengers traveled in Third Class.
- Passenger ages were concentrated between 20 and 40 years.
- Ticket fares were positively skewed, with a few passengers paying significantly higher fares.
